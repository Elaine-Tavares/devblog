O **DevBlog** não é apenas mais um blog; é uma Single Page Application (SPA) robusta, projetada para ser um ponto de encontro entre desenvolvedores. Este projeto demonstra a integração fluida entre um frontend moderno e reativo com um backend sólido, focando em performance, experiência do usuário (UX) e facilidade de gerenciamento.

---

## 🌟 Por que este projeto merece sua atenção?

Como desenvolvedor, meu objetivo foi criar uma solução completa que unisse a agilidade do **React** com a confiabilidade do **PHP**. O DevBlog resolve o desafio de gerenciar conteúdos dinâmicos com uma interface intuitiva e funcionalidades que engajam o usuário desde o primeiro segundo.

### ✨ Destaques de Funcionalidade:
- **Painel Administrativo Completo:** Um dashboard exclusivo para gestão total de posts (Create, Read, Update).
- **Engajamento Interativo:** Um quiz personalizado na Home que identifica o perfil do desenvolvedor (Front-end, Back-end ou Fullstack).
- **Gestão de Mídia Inteligente:** Suporte para uploads de imagens e documentos, permitindo posts ricos em conteúdo técnico.
- **Sistema de Comentários:** Fomenta a comunidade através da troca de feedbacks em tempo real.
- **Arquitetura Escalável:** Organização de pastas por funcionalidades (pages/components) e centralização de APIs com Axios.

---

## 🛠️ Stack Tecnológica

### Frontend
- **React 18 & Vite:** O estado da arte em termos de velocidade de build e reatividade.
- **React Router Dom:** Gerenciamento de rotas complexas e navegação fluida sem recarregamento.
- **CSS Modules:** Estilização isolada e segura, evitando conflitos de escopo.
- **React Icons:** Interface polida com iconografia moderna.
- **Axios:** Comunicação assíncrona otimizada com o backend.

### Backend (API)
- **PHP:** Processamento robusto de dados e lógica de negócio.
- **MySQL (via PDO):** Persistência de dados segura e eficiente.
- **Filesystem Management:** Sistema customizado para armazenamento e recuperação de uploads.

## 🚀 Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/devblog.git
   ```

2. **Instale as dependências do Frontend:**
   ```bash
   cd frontend
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Configuração do Backend:**
   Certifique-se de que o ambiente PHP (XAMPP/WAMP ou servidor remoto) esteja rodando e a `baseURL` em `src/axios/api.js` esteja apontando corretamente para o seu servidor.
